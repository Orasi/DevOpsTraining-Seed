PUT /environments/:id
Update existing environment

Only accessible if current user can view parent project

Examples

PUT /environments/10374
{
  "environment": {
    "uuid": "987654321",
    "display_name": "Some new display name"
  }
}
200
{
  "environment": {
    "project_id": 2082,
    "id": 10374,
    "uuid": "987654321",
    "display_name": "Some new display name",
    "environment_type": "Undefined",
    "created_at": "2017-05-17T16:26:12.781Z",
    "updated_at": "2017-05-17T16:26:13.178Z"
  }
}
