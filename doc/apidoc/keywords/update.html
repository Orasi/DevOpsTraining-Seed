PUT /keywords/:id
Update existing keyword

Only accessible if current user can view parent project. Pass optional array of testcase ids to associate keyword to tests

Examples

PUT /keywords/6313
{
  "keyword": {
    "keyword": "MORE AWESOME"
  },
  "testcases": [
    10536,
    10537,
    10538,
    10539,
    10540
  ]
}
200
{
  "keyword": {
    "project_id": 2106,
    "id": 6313,
    "keyword": "MORE AWESOME",
    "created_at": "2017-05-17T16:26:24.670Z",
    "updated_at": "2017-05-17T16:26:24.995Z",
    "testcase_count": 5
  }
}
