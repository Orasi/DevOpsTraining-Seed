GET /results/:id/screenshot/:screenshot_id
Access result screenshot

Returns a temporary url to access the screenshot directly. \

Includes a screenshot token that is valid for 30 seconds from \
time of creation and destroyed after first use.  Only accessible \
if parent project is viewable by current user

Examples

GET /results/18489/screenshot/18
200
{
  "screenshot": "http://example.org/download/D9p4HnRr3VyRqVgsCZyMC8DP"
}

Params

Param name Description
User-Token
required

Authentication token. Must be provided in the header of all calls unless otherwise indicated

Validations:

  • Must be String
id
required

Result ID

Validations:

  • Must be a number.
screenshot_id
required

Screenshot ID

Validations:

  • Must be a number.
