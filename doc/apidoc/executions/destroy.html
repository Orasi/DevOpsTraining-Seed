DELETE /executions/:id
Delete execution

Deletes an existing execution

Metadata

--- |-
  If deleted execution was open it will create a new open execution
          Projects must have one open execution at all times
          Only accessible by Admins

Examples

DELETE /executions/2145
200
{
  "execution": "Deleted"
}

Params

Param name Description
User-Token
required

Authentication token. Must be provided in the header of all calls unless otherwise indicated

Validations:

  • Must be String
id
required

Execution ID

Validations:

  • Must be a number.
