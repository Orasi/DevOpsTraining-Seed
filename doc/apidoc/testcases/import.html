POST /projects/:id/import
Import testcases

Import testcases from CSV string. Only accessible by admin users

Examples

POST /projects/24544/import
{
  "csv": "Test ID,Title,Steps,Expected Result\n123457,New Test Case 1,\"1.  Action 1\n2.  Action 2\n3.  Action 3\",\"1.  Result 1\n2.  Result 2\n3.  Result 3\"\n123456,New Test Case 2,\"1.  Action 1\n2.  Action 2\n3.  Action 3\",\"1.  Result 1\n2.  Result 2\n3.  Result 3\"\n"
}
200
