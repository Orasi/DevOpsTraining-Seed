POST /users/reset-password
Trigger Password Reset Email

Triggers a password reset email to the User

Supported Formats

json

Examples

POST /users/reset-password
{
  "user": {
    "username": "eleazar_langworth@howell.io"
  },
  "redirect-to": "http://some.url/TOKEN"
}
200
{
  "success": "Password reset email sent"
}

Params

Param name Description
User-Token
required

Authentication token. Must be provided in the header of all calls unless otherwise indicated

Validations:

  • Must be String
id
required

Username

Validations:

  • Must be String
