GET /results/:id/screenshot/:screenshot_id
Access result screenshot

Returns a temporary url to access the screenshot directly. \

Includes a screenshot token that is valid for 30 seconds from \
time of creation and destroyed after first use.  Only accessible \
if parent project is viewable by current user

Examples

GET /results/95368/screenshot/184
200
{
  "screenshot": "http://example.org/download/TZhbdgUw3pxJBPAv8jvW9Vm3"
}
