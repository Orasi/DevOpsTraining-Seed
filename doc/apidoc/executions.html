POST /executions/close
Close execution

Closes execution and opens a new execution

Metadata

--- |-
  User must provide either execution_id or project_key
          If project_key is provided the open execution will be closed
          Only accessible if project is viewable by current user
          Displays details of NEW execution

Examples

POST /executions/close
{
  "project_key": "eb96e9df1768e21734b667ac848d2a75",
  "name": "Some Name"
}

{
  "execution": {
    "id": 2142,
    "created_at": "2017-05-17T16:26:22.573Z",

}

Param name Description
