Resources

Application

Resource Description

Downloads

Resource Description

Environments

Resource Description
GET /environments/:id Environment details
POST /environments Create new environment
PUT /environments/:id Update existing environment
DELETE /environments/:id Delete existing environment

Resource Description
get /executions/:id/testcase-count Testcase Count

GET /executions/:id/testcase_status Execution Details
GET /executions/:id/testcases/:testcase_id Testcase Detail
GET /executions/:id/environments/:environment_id Environment Detail
GET /executions/:id/incomplete_tests List Incomplete Tests
POST /executions/close Close execution
DELETE /executions/:id Delete execution
GET /executions/:id/next_test Next Incomplete Test
GET /executions/:project_key/failing Get all failing tests

Projects

Resource Description

Results

Resource Description

Teams

Resource Description
GET /teams All teams
GET /teams/:id Team details
POST /teams Create new team
PUT /teams/:id Update existing team
DELETE /teams/:id Delete existing team
POST /teams/:id/user/:user_id Add existing user to team
POST /teams/:id/projects/:project_id Add existing project to team
DELETE /teams/:id/user/:user_id Remove user from team
DELETE /teams/:id/projects/:project_id Remove project from team

Resource Description

PUT /users/:id Update existing user
DELETE /users/:id Delete existing user
