

# Mustard
Ruby on Rails based multi-environment test results server.  Mustard collects test results from closely related tests (IE Cross Browser or Mobile Testing on Multiple Device) and groups them for easy analysis.  The Mustard-See is the backend server and can be combined with [Mustard-Dijon](https://github.com/Orasi/Mustard-Dijon) for viewing test results easily.
